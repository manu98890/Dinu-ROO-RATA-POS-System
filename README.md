# Dinu Roo Rata POS System

Meya oya illapu "Dinu Roo Rata" POS system ekayi.

## Patan Ganna Widihata (How to Start)

1. **Internet Connect Wenna**: First time open karaddi witarak internet ona wei (Tailwind CSS, Icons, saha Database library eka ganna).
2. `index.html` file eka uda **Double Click** karanna.
3. Web browser eken POS system eka open wei.

## Features

1. **Dashboard**: Ada dawase sales, profit, saha badu iwara wenna yana ewa balaganna puluwan.
2. **POS (Point of Sale)**:
   - Badu search karanna puluwan (Name ho Code nethnam Category eken).
   - Cart ekata add karala Bill eka hadanna puluwan.
   - Checkout kalama Stock adu wena gaman Sales record ekak save wenawa.
3. **Inventory**:
   - Aluth badu add karanna, thiyena ewa edit/delete karanna puluwan.
   - Stock manage karanna puluwan.
4. **Reports**:
   - Kalin wikunu ewa balaganna puluwan.
   - Profit eka bala ganna puluwan.
   - **Export CSV/PDF**: Sales data report ekak widihata download karanna puluwan.

## Technical Note
- Meya **Offline** weda karanawa data save karanna (indexedDB use karala).
- Namuth UI design eka ganna (Tailwind CSS) saha Icons ganna (Lucide) internet connection eka initial load ekata awashyai.
